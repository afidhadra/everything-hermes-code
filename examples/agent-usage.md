# Agent Usage Examples

Examples for using different AI agents in the toolkit.

## Architect Agent

**Purpose:** System design and architecture decisions.

**Usage:**

```text
Act as architect agent: Design a microservices architecture for an e-commerce platform.
```

**Example Output:**

- System components
- Data flow diagrams
- Technology stack recommendations
- Scalability considerations

## Coder Agent

**Purpose:** Code implementation and best practices.

**Usage:**

```text
Act as coder agent: Implement a REST API endpoint for user authentication.
```

**Example Output:**

- Clean code implementation
- Error handling
- Input validation
- Documentation
