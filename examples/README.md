# Examples

## Overview

Example setups and configurations.

## Available Examples

| Example | Directory | Description |
|---------|-----------|-------------|
| Basic Setup | basic/ | Minimal configuration |
| Full Setup | full/ | Complete configuration |
| Project Setup | project/ | Per-project configuration |

## Basic Setup

Minimal configuration for getting started.

```bash
# Copy to ~/.hermes/
cp -r examples/basic/* ~/.hermes/
```

## Full Setup

Complete configuration with all features.

```bash
# Copy to ~/.hermes/
cp -r examples/full/* ~/.hermes/
```

## Project Setup

Per-project configuration.

```bash
# Copy to project root
cp -r examples/project/* ~/Projects/my-project/
```

## Creating Examples

1. Create a new directory in this directory
2. Add configuration files
3. Add a README.md explaining the example
4. Test the example before committing
